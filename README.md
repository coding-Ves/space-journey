
## What's this?
A quick project to practice Three.js and working with 3D models, inspired by and partly following https://www.youtube.com/watch?v=WL5zrKii5qc&ab_channel=WawaSensei 

## How do I run this?

```
npm install 
```

```
npm run dev
```
